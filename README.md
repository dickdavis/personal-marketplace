# personal-marketplace

This repository serves as a personal marketplace for my custom Claude Code plugins.

## Usage

This marketplace is already configured if you are using the [settings](https://github.com/dickdavis/dotfiles/blob/master/templates/.claude/settings.json) from my [dotfiles](https://github.com/dickdavis/dotfiles/tree/master) repo.

To manually add the marketplace from within Claude Code:

```
/plugin marketplace add dickdavis/personal-marketplace
```

You can then browse all of the available plugins.

## Plugins

* [rails-plugin](https://github.com/dickdavis/rails-plugin) - Claude Code plugin for building Rails applications.
* [technical-writer-plugin](https://github.com/dickdavis/technical-writer-plugin) - Claude Code plugin for writing effective technical documentation.
* [terminal-security-plugin](https://github.com/dickdavis/terminal-security-plugin) - Claude Code plugin that provides an extra layer of defense and auditability.
